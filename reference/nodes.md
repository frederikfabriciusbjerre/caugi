# Get nodes or edges of a `caugi`

Get nodes or edges of a `caugi`

## Usage

``` r
nodes(cg)

vertices(cg)

V(cg)
```

## Arguments

- cg:

  A `caugi` object.

## Value

A `data.table` with a `name` column.

## Examples

``` r
cg <- caugi(
  A %-->% B,
  B %-->% C,
  D,
  class = "DAG"
)
nodes(cg) # returns the data.table with nodes A, B, C, D
#>      name
#>    <char>
#> 1:      A
#> 2:      B
#> 3:      C
#> 4:      D
```
